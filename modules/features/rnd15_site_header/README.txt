The site header of rnd15 is handled by the *rnd15_site_header* feature. It provides a context that sets the blocks to be provided in the header. These are currently:

* ESU Header
* Navigation

The main styles are located in the rnd15 Theme folder under 
<code>sass/components/_header.scss</code>
