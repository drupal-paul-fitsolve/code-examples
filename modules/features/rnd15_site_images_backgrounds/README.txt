The Site Images Backgrounds module provides picture groups with predefined cropping points and exposes it as an option in crl2 panel style section plugin.

For more information on the Background crops please refer to the Confluence documentation:
http://confluence.comicrelief.com/display/rnd15EditorDocumentation/How+to+set+a+Section+Background

Process - Set a Pane Picture Group:
--------

Setting the Background Pane Picture Group:
1: Set a section to use pane backgrounds by ticking "Use Background Pane"

2: Within the Background Section, select the Pane Picture group you wish the pane to use

3: Hit save and save the panel page


Process - Creating a new Background Pane Picture Group:
--------
1: Create a set of image styles

2: Create a picture group but ensure it has the word "focus" or "crop_width" in it.

3: The new picture group will appear in the section background settings section
