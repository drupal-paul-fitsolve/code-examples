The site header of sr16splash is handled by the *sr16splash_site_header* feature. It provides a context that sets the blocks to be provided in the header. These are currently:

* ESU Header
* Navigation

The main styles are located in the sr16splash Theme folder under 
<code>sass/components/_header.scss</code>
